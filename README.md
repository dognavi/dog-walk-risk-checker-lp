# 散歩危険度チェッカーLP

愛犬との散歩に潜む危険（熱中症・アスファルト熱・紫外線・交通量など）を数値化して知らせる「散歩危険度チェッカー」ツールを紹介するランディングページ（LP）。
訪問者が「今すぐ使ってみたい」と感じるビジュアル・コピー・インタラクションを重視したデザインで構成する。

## コンセプト

- **課題提起**: 真夏の散歩は人が思う以上に犬にとって危険（アスファルト温度、熱中症、紫外線など）だが、飼い主が体感で気づきにくい。
- **解決策**: 気温・湿度・時間帯などから「今の散歩危険度」をスコア化して一目で見せるツール。
- **LPの役割**: ツール本体への導線となる紹介ページ。ファーストビューで危険度を疑似体験させ、CTA（今すぐチェックする）へ自然に誘導する。

## セクション構成（実装済み）

1. ヒーロー（メインコピー＋サブコピー＋CTAボタン＋マイクロコピー、犬のイラストを上に大きく配置）
2. なぜチェックが必要か（湿度・地面との距離・犬が我慢してしまう性質という3つの気づき）
3. このツールで何がわかるか（便益訴求＋3ステップ＋ツールのスクリーンショット）
4. 締め（コピー再訴求＋CTAボタン再掲、ヒーローと呼応する専用イラスト）

CTAボタンは`https://dognavi.github.io/dog-walk-risk-checker/?ref=lp`（ツール本体、`ref=lp`はLP経由の流入判別用）に新規タブで遷移する。

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript（ビルドツールなしの静的サイト）
- アニメーション: CSSトランジション／IntersectionObserverによるスクロール演出
- デプロイ想定: 静的ホスティング（GitHub Pages, Vercel, Netlify 等）

ビルドステップを持たないシンプルな静的LPのため、Node/Python等のランタイムには依存しない。

## ディレクトリ構成

```
.
├── index.html      # LP本体
├── css/
│   └── style.css   # スタイル定義
├── js/
│   └── script.js   # スクロール時のフェードイン演出（IntersectionObserver）
└── images/
    ├── header.png     # ヒーロー用イラスト（散歩する犬）
    ├── footer.png     # 締め用イラスト（正面を向いて笑う犬）
    └── screenshot.png # ツール本体のスクリーンショット
```

## 開発

ビルド不要。`index.html` をブラウザで直接開くか、ローカルサーバーを使う。

```bash
# 例: Python標準ライブラリで簡易サーバーを立てる場合
python -m http.server 8000
```
